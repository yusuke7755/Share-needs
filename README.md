---

実行手順
---
```
$ git clone git@github.com:yusuke7755/Share-needs.git
$ cd Share-needs
$ yarn add jquery bootstrap@4
$ bundle install
$ rails db:create
$ rails db:migrate
$ rails db:seed
$ rails s
```
